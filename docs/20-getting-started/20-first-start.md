# Подготовка к первому запуску

1. **Распакуйте устройство**
   - Достаньте Робоголову из упаковки
   - Проверьте комплектацию (см. раздел [Комплектация](../10-introduction/20-equipment.md))
   - Убедитесь в отсутствии видимых повреждений

2. **Установите подставку** (если не установлена)
   - Закрепите Т‑гайки в пазах подставки
   - Прикрутите устройство винтами `М5×12` (входят в комплект)
   - Убедитесь в устойчивости конструкции

3. **Подключите кабель питания и кабель термистора к аккумуляторной сборке**
   - Открутите 4 винта на верхней части Робоголовы

   ![Необходимые винты](attachements/robohead_4m3.JPG)

   - Аккуратно приподнимите верхнюю крышку
   - Подключите красно‑черный кабель питания к разъему **+|-** аккумуляторной сборки
   - Подключите синий кабель термистора к разъему **TH** аккумуляторной сборки

   ![Подключение проводов](attachements/robohead_cable.JPG)

   - Установите крышку обратно и закрутите винты

# Процесс включения

1. **Подключите питание**
   - Используйте оригинальный блок питания `12В/2.5А` (DC-Jack)
   - Вставьте кабель в разъём на задней панели устройства

:::note
*Используйте только оригинальное зарядное устройство!*
:::

2. **Запуск системы**
   - Нажмите кнопку питания (расположена сзади Робоголовы)
   - Дождитесь подсветки микрофонного модуля (сверху Робоголовы)
   - На дисплее появится загрузочный экран

3. **Загрузка устройства**
   - Процесс занимает **около минуты**
   - Во время загрузки экран может кратковременно гаснуть
   - После завершения на экране отобразится стандартное «лицо»

![Изображение в режиме ожидания](attachements/wait.png)

   - Динамики воспроизведут приветственное сообщение:  
     > «Ну привет»

# Проверка работоспособности

## Тест голосового управления

- Чётко произнесите: **"Слушай, Робот!"**
- Устройство должно:
  - Повернуться к вам
  - Изменить выражение "лица"
  - Подать звуковой сигнал готовности: **"Чего надо?"**

## Проверка базовых команд

Попробуйте произнести:

- **"Поздоровайся"** – робот произнесёт приветствие  
- **"Покажи уши"** – сервоприводы «ушей» придут в движение  
- **"Сделай фото"** – камера сделает снимок

## Возможные проблемы

| Симптом                         | Решение                                                                                   |
|---------------------------------|-------------------------------------------------------------------------------------------|
| Нет реакции на кнопку питания   | Проверьте подключение кабеля, попробуйте другой блок питания                              |
| Дисплей не включается           | < описание перехода в режим тестирования >                                                  |
| Нет звукового сигнала           | < описание переподключения USB в Raspberry Pi >                                            |
| Не распознаёт команды           | Говорите чётко на расстоянии 0.5–1.5 м, уменьшите фоновый шум                            |

## Советы для первого использования

- Работайте в тихом помещении при первом тестировании
- При низком заряде батареи подключите питание (см. раздел [Контроль заряда аккумулятора](../10-introduction/10-main-features.md/#-контроль-заряда-аккумулятора))


> После успешного первого включения рекомендуем сразу подключиться к Wi‑Fi и обновить программное обеспечение.
