id: 32-changing-password
title: Изменение пароля
---

Напомним стандартные учетные данные:

```text
### Стандартные учетные данные
- Логин: pi
- Сетевое имя: roboheadXXX
- Пароль по умолчанию: brobro (рекомендуется сменить после первого входа)
```

Для смены пароля вам необходимо знать текущий пароль или иметь права `sudo`.

---

## Шаги изменения пароля

1. **Подключитесь к устройству по SSH:**

   ```bash
   ssh pi@roboheadXXX.local
   ```
   
2. **Запустите утилиту `passwd`:**

   ```bash
   passwd
   ```

   :::note
   При вводе пароля символы не отображаются — это мера безопасности.
   :::

3. **Следуйте подсказкам:**

   * Введите текущий пароль.
   * Введите новый пароль.
   * Подтвердите новый пароль.

## Требования к новому паролю

* **При использовании без `sudo`:** минимум 6 символов, избегайте слишком простых комбинаций.
* **Рекомендуемые опции:**

  * 8+ символов
  * Заглавные и строчные буквы
  * Цифры и спецсимволы
  * *Пример:* `R0b0head#2024`

## Альтернативный метод автоматической смены пароля

Вы можете сменить пароль одной командой (при наличии прав `sudo`):

```bash
echo "pi:новый_пароль" | sudo chpasswd
```

## Проверка изменений

Убедитесь, что новый пароль применился:

```bash
su - pi  # введите новый пароль
```

## Возможные ошибки и решения

| Ошибка                                  | Причина                  | Решение                                       |
| --------------------------------------- | ------------------------ | --------------------------------------------- |
| Authentication token manipulation error | Недостаточно прав        | Используйте `sudo passwd` или проверьте права |
| BAD PASSWORD                            | Пароль слишком простой   | Выберите более сложный пароль                 |
| Permission denied                       | Отсутствуют права `sudo` | Выполните смену через `sudo chpasswd`         |
