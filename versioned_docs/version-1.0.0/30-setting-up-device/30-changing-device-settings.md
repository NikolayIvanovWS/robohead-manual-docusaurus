**Робоголова Bbrain** позволяет гибко настраивать параметры устройства под ваши задачи. В этом разделе описаны основные настройки, которые можно изменить:

## Доступные параметры для изменения

1. **Сетевые настройки**

   * Сетевое имя устройства (*hostname*)
   * Пароль для доступа по SSH
   * Настройки Wi‑Fi подключения

2. **Аудиопараметры**

   * Громкость динамиков

3. **Механические настройки**

   * Инициализационное положение сервоприводов:

     * шеи
     * ушей
   * Ограничения на углы поворота сервоприводов

4. **Визуальные параметры**

   * Подсветка микрофонного модуля (цвет, яркость)

---

## Как изменять настройки?

Практически все параметры можно изменить, отредактировав соответствующие файлы в папке `config` пакета `robohead/robohead_controller`.
